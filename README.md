# 🤖 AI Job Mentor for Informal Workers

A multi-agent AI-powered career guidance platform built for 
India's informal workforce — daily wage laborers, artisans, 
gig workers, and self-employed individuals.

Built using **IBM Watsonx Orchestrate**, **IBM Granite Models**, 
and **IBM Cloud Lite** as part of the IBM & AICTE Agentic AI 
program.

## 🎯 Problem Statement
Millions of informal workers in India lack access to:
- Relevant job opportunities
- Knowledge of government schemes (PMKVY, MUDRA, NRLM)
- Skill development resources
- Personalized career guidance

This platform bridges that gap using Agentic AI.

---

## ⚡ Features
- 🔍 Personalized job opportunity suggestions
- 🏛️ Government scheme recommendations (PMKVY, MUDRA, NRLM)
- 📚 Skill development course guidance
- 🌍 Vernacular language support
- 🤖 Multi-agent AI architecture
- 📱 Fully responsive UI
